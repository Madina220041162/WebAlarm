# Alarm Sound Files

Place your alarm sound files in this directory with the following names:

## Required Files:
- `rooster.mp3` - Rooster crowing sound
- `heavy-metal.mp3` - Heavy metal guitar riff
- `military-trumpet.mp3` - Military trumpet/bugle call
- `classic-clock.mp3` - Classic alarm clock ringing
- `electronic-beep.mp3` - Electronic beep sound

## Where to Get Free Sounds:

### Option 1: Freesound.org (Recommended)
1. Visit https://freesound.org
2. Search for each sound type
3. Download MP3 format
4. Rename files to match the names above

### Option 2: Zapsplat.com
1. Visit https://www.zapsplat.com
2. Free account required
3. Search and download sounds

### Option 3: YouTube Audio Library
1. Visit https://www.youtube.com/audiolibrary
2. Search for alarm sounds
3. Download and convert to MP3

### Option 4: Use AI Sound Generators
1. Visit https://elevenlabs.io/sound-effects
2. Generate custom alarm sounds
3. Download as MP3

## Recommended Search Terms:
- **Rooster**: "rooster crow", "rooster morning", "cock a doodle doo"
- **Heavy Metal**: "metal guitar riff", "distorted guitar", "heavy metal alarm"
- **Military Trumpet**: "bugle call", "reveille", "military trumpet"
- **Classic Clock**: "alarm clock ring", "mechanical alarm", "bell alarm"
- **Electronic Beep**: "digital alarm", "beep alarm", "electronic alert"

## File Requirements:
- Format: MP3 (recommended) or WAV
- Duration: 3-10 seconds (will loop automatically)
- Volume: Normalized/medium volume
- Sample Rate: 44.1kHz or 48kHz

## Note:
If sound files are not found, the system will automatically fallback to generated Web Audio API sounds.
